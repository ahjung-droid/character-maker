# 🌸 우리 회사 부캐 메이커

신규 입사자 워크샵용 성향 분석 & 캐릭터 매칭 웹앱입니다.
7개의 질문에 답하고 사진을 업로드하면, AI가 성향을 분석해
귀여운 동물 캐릭터와 맞춤형 별명을 생성해줍니다.

## ✨ 주요 기능

- 7개 밸런스 게임 스타일 질문으로 성향 분석
- 6종의 동물 캐릭터(리트리버/너구리/쿼카/고양이/판다/토끼) 매칭
- 참가자마다 고유한 별명 자동 생성 (중복 방지)
- 업로드 사진 기반 결과 카드 PNG 다운로드
- 관리자 페이지(비밀번호 인증)에서 참가자 전체 관리
- Midjourney/DALL-E용 이미지 생성 프롬프트 자동 생성
- 관리자용 JSON 데이터 내보내기

## 🚀 사용 방법

`index.html` 파일을 브라우저에서 열면 바로 동작합니다.
별도의 서버나 빌드 과정이 필요 없어요.

## 🔑 관리자 접속

오른쪽 상단의 "관리자" 버튼 → 4자리 비밀번호 입력
비밀번호는 `index.html` 상단의 `ADMIN_PASSWORD` 상수에서 변경 가능합니다.

## 📦 데이터 저장

모든 참가자 결과는 브라우저의 localStorage에 저장됩니다.
별도 서버로 전송되지 않아 개인정보 걱정이 없어요.
